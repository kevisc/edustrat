Educational Inequality Overview

This dashboard provides empirical analysis of educational inequalities using standardized assessment data. The analysis framework draws on social stratification theory (Bourdieu & Passeron, 1977; Coleman et al., 1966) to examine how socioeconomic factors correlate with student achievement outcomes.

Key Metrics Explained:
Inequality Index (Gini): Measures the dispersion of achievement scores within the selected sample (0 = perfect equality, 1 = maximum inequality). This reflects how evenly distributed test scores are among students, independent of the number of countries. A higher Gini coefficient indicates greater inequality in achievement.
SES Gradient (β): The slope coefficient indicating how much achievement increases per unit increase in socioeconomic status. A higher value indicates stronger SES-achievement association.
Mean Achievement: Average test score across all selected students and countries.

Methodology

This application implements established methods from educational sociology and stratification research to analyze the relationship between socioeconomic background and educational achievement using PISA data.

Key Methods Implemented

  • Survey-Weighted Analysis: Accounts for complex sampling design using student weights and replicate weights
  • Regression Models: OLS regression with controls, multilevel models for cross-country analysis
  • Inequality Metrics: Gini coefficient, Lorenz curves, Theil index, achievement gaps
  • Decomposition: Variance decomposition and Blinder-Oaxaca decomposition for temporal analysis
  • Within/Between Analysis: Separates within-country and between-country variation
